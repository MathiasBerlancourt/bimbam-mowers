# 🚜 bimbam-mowers 🚜

## 📝 Description

This project is a technical recruitment test with React.

## ⚙️ Installation steps

## Using npm

**Run commands**

1. `npm install`

2. `npm run dev`

## Or using yarn

**Run commands**

1. `npm install --global yarn`

2. `yarn install`

3. `yarn run dev`

**Steps to run the app**

- Upload the .txt file "test.txt" (sample available on project root)
- Run the steps
